# Draftigo

Draftigo is an AI-powered email assistant that helps users generate professional and context-aware email replies instantly. Users can paste an email, choose the desired tone, and receive an AI-generated response, improving productivity and communication efficiency.

## Features

- AI-generated email replies
- Multiple response tones (Professional, Friendly, Formal, etc.)
- Clean and intuitive user interface
- Real-time response generation
- REST API integration between frontend and backend
- Fast and responsive user experience

## Tech Stack

### Frontend
- JavaScript
- React.js
- Vite
- Material UI (MUI)

### Backend
- Java
- Spring Boot
- Spring Web
- REST APIs

### AI Integration
- Generative AI API
- Prompt Engineering
